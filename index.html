Printing instructions

  • A ZIP file will be created, it contains 4 pieces generated to your dimensions and a file for the nail.

  • Add the 4 pieces in your 3D printer software. Duplicate the nail 3 times.

  • Simplify3D will show an alert about the scale of the nail being wrong, you can ignore it (do not resize).
    If you need to change the size of the nail manually, it's a rectangle with a length of 9mm and a side of 2.7mm

  • This print requires a layer height of 0.2mm or less

  • You can print it as hollow as 8%, below that prints might fail.

  • Use 5 top and bottom layers, and 5 perimeter shells

  • Temperature is quite important here as the tolerances are very small.
    Overheating the filament / bed will result in distortion, and pieces might not fit corrrectly.
    I get good results when using PLA at 190C with hairspray on glass (without bed heating).

  • Print this set twice.

  • When removing the pieces from the bed, lift them from their square side (the angled corners are more fragile).

  • Make sure you have the assembly ready before adding the nails. Once put in place, you won't be able to remove them.
